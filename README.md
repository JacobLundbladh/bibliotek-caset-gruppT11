# Library Management System

This project is a distributed library system built using ASP.NET MVC and multiple Web APIs.

## Architecture

The system consists of:

- MVC Web Application (Client)
- User API
- Loan API
- Item API
- Reminder API

Each service is deployed separately on Azure and communicates using HTTP.

## Features

- User authentication
- Borrow and return items
- Manage loans
- Reminder system for overdue loans

## Technologies

- ASP.NET Core MVC
- Web APIs
- Entity Framework Core
- Azure App Services
- HTTPClient communication between services

## Purpose

This project was developed as part of a university course to demonstrate service-based architecture and API communication.
